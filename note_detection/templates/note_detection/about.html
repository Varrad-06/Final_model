About This Project

Counterfeit Indian Currency Note Detection System

This system is designed to detect counterfeit Indian 500 rupee notes using image processing and computer vision techniques. The detection is based on analyzing various security features present in genuine Indian currency notes.

Key Features

  • Visual Security Features Analysis: Compares 7 distinct visual security elements against templates of genuine notes.
  • Bleed Lines Detection: Analyzes the security bleed lines on both left and right sides of the note.
  • Number Panel Verification: Verifies the number panel by detecting all 9 characters that should be present.

How It Works

  1. Upload a clear image of an Indian 500 rupee note.
  2. The system preprocesses the image (resizing, blurring, grayscale conversion).
  3. Features are extracted and compared with templates of genuine notes.
  4. The results are analyzed to determine if the note is authentic or suspicious.

Technology Used

  • Backend: Django (Python web framework)
  • Computer Vision: OpenCV, NumPy, scikit-image
  • Feature Detection: ORB (Oriented FAST and Rotated BRIEF)
  • Similarity Measurement: Structural Similarity Index (SSIM)
  • Frontend: HTML, CSS, Bootstrap, JavaScript
Disclaimer: This system is for educational purposes only. It should not be used as a definitive tool for currency verification in real-world financial transactions.
